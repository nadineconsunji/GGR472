Project Methods

Overview of data, design process, and interactivity

Data

The project utilised data from CATF detailing Nigeria’s existing clean technologies and capacity to achieve a clean energy transition through three numerical scores:

  • System Performance – Measures economic development, sustainability (CO₂ output, biodiversity), and energy access/security.
  • Transition Readiness – Assesses governance, political commitment, human capital, investment, and power systems.
  • Composite Index – Combines the above with existing clean technologies into a single score.

Each score ranges from 0–100, where higher values indicate greater transition capacity.

Because CATF is still collecting continental data, we generated dummy data for other African countries using randomized values. These were merged with a GeoJSON file of African country boundaries to enable mapping.

Design

Our primary goal was to visualize clean energy transition scores using an accessible and visually coherent colour ramp. We aligned colours with CATF’s branding while ensuring compatibility with the basemap.

We prioritised intuitive design through clear visual hierarchy and strategic placement of information buttons.

Workflow
  1. Data collection & dummy data creation
  2. Baseline continental map
  3. Country-level development
  4. Popup content
  5. User interface improvements
  6. Final layout & design

Weekly meetings were used to troubleshoot, assign tasks, and review progress. Development included collaborative coding practices such as branch management and merge discussions.

Feedback from testing and peer review (including CATF) led to improvements in colour schemes, formatting, interactivity clarity, and the addition of informational tools.

Interactivity

The web map includes a range of interactive features to enhance usability:

  • Zoom controls, geocoder, fullscreen mode
  • Clickable CATF logo linking to homepage
  • Layer toggle for different transition metrics
  • Dynamic legend that updates with selected data

Users can also explore renewable readiness scores (e.g., hydrogen, CCS) in a separate tab.

Map Interactions
  • Hover: View country-level scores
  • Double-click: Zoom + detailed country summary
  • Region filter: Focus on specific areas
  • Custom selection: Calculate regional averages

A collapsible sidebar allows users to maximise map visibility and focus.
